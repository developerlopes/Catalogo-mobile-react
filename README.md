
# Catálogo Mobile 

Projeto em React Native com Expo SDK 54

## Requisitos atendidos
- Tela de login com validação
- Armazenamento temporário com Redux Toolkit
- Navegação por abas/categorias masculinas e femininas
- Consumo de API com Axios
- Tela de detalhes do produto
- Logout funcional

## Categorias implementadas
### Masculino
- mens-shirts
- mens-shoes
- mens-watches

### Feminino
- womens-bags
- womens-dresses
- womens-jewellery
- womens-shoes
- womens-watches

## Como executar
```bash
npm install
npx expo install
npx expo start --tunnel
```

## API
- https://dummyjson.com/products/category/{categoria}
- https://dummyjson.com/products/{id}
